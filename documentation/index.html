Quick Start

Transform a local script into a distributed task in seconds:

from slurmray import RayLauncher

# 1. Define your logic
def compute_heavy_pi(n):
    # This code runs on the remote cluster nodes!
    return 3.1415926535

# 2. Configure the bridge
launcher = RayLauncher(
    project_name="pi_calc",
    cluster="curnagl",
    node_nbr=4,
    use_gpu=True
)

# 3. Execute!
result = launcher(compute_heavy_pi, args={"n": 10**8})
